# md-spring-mongodb

## Description
Simple CRUD project using MongoDB.

## Technologies
- Java 21
- Spring Boot 3.5.0
- Maven
- MongoDB
- Swagger (OpenAPI) for API documentation

## Requirements
- Java 21 installed
- Maven 3.9.x or higher installed

## Build and Run

### Build the project
`mvn clean install`
### Run the project
`mvn spring-boot:run`
